Q 1 Define Inferential Statistics and Descriptive Statistics. 3 M

Descriptive Statistics

Definition: Descriptive statistics summarizes and describes the main features of a dataset using numerical measures and graphs. It does NOT draw conclusions beyond the data itself.

  • Measures of Central Tendency: Mean, Median, Mode
  • Measures of Dispersion: Range, Variance, Standard Deviation
  • Examples: "Average marks of a class = 72" or a histogram showing marks distribution

Inferential Statistics

Definition: Inferential statistics uses a sample from a population to draw conclusions (inferences) or make predictions about that larger population.

  • Key Tools: Hypothesis testing, confidence intervals, regression, p-values
  • Example: Surveying 500 students from a university to estimate average CGPA of all 10,000 students
Descriptive vs Inferential Statistics

Q 2 Define Probability. Explain characteristics of probability. 4 M
Sum'24Sum'25

Definition

Probability is a numerical measure of the likelihood (chance) that an event will occur. It is expressed as a number between 0 and 1.

P(Event) = Number of favorable outcomes / Total number of possible outcomes

Example: P(Head on a coin) = 1/2 = 0.5

Characteristics of Probability

  1. Range [0, 1]: Probability of any event lies between 0 (impossible) and 1 (certain). P = 0 means the event cannot occur; P = 1 means it will always occur.
  2. Complementary Rule: P(A) + P(A') = 1. The probability of an event and its complement always sum to 1.
  3. Addition Rule: For mutually exclusive events, P(A or B) = P(A) + P(B).
  4. Multiplication Rule: For independent events, P(A and B) = P(A) × P(B).
  5. Non-negativity: Probability is always ≥ 0. It is never negative.
  6. Sum of all outcomes: The sum of probabilities of all possible outcomes equals 1.

Types of Probability

Classical

Based on equally likely outcomes. E.g., rolling a die: P(4) = 1/6

Empirical

Based on historical data / experiments. E.g., past sales records.

Subjective

Based on personal judgment or expert opinion.

Conditional

P(A|B) — probability of A given B has occurred.

What are Outliers?

An outlier is a data point that differs significantly from other observations in the dataset. Outliers can be caused by measurement errors, data entry mistakes, or genuinely rare events.

Impact: Outliers skew the mean, inflate variance, distort regression lines, and reduce model accuracy. Hence they must be detected and handled before analysis.

Outlier Detection Techniques

1. Z-Score Method

Z = (X - μ) / σ

If |Z| > 3, the data point is considered an outlier. Works best for normally distributed data.

2. IQR (Interquartile Range) Method

IQR = Q3 - Q1
Lower bound = Q1 - 1.5 × IQR
Upper bound = Q3 + 1.5 × IQR

Any value outside [Lower bound, Upper bound] is an outlier. This is a robust method that does not assume normal distribution.

3. Box Plot (Visual Method)

A box plot displays the five-number summary (Min, Q1, Median, Q3, Max). Data points plotted beyond the whiskers are outliers.

Box Plot — Outlier Detection

4. Scatter Plot (Visual)

Points far from the main cluster in a scatter plot are outliers.

5. DBSCAN Clustering

Points that do not belong to any cluster are labeled as outliers (noise points).

What is Correlation?

Correlation is a statistical measure that describes the strength and direction of a linear relationship between two variables. It ranges from –1 to +1.

Positive Correlation (+1)

Both variables increase together. E.g., Height and Weight.

Negative Correlation (–1)

One increases while the other decreases. E.g., Temperature and Heater usage.

Zero Correlation (0)

No linear relationship between the variables.

Range

–1 ≤ r ≤ +1. Closer to ±1 = stronger; closer to 0 = weaker.

Methods to Calculate Correlation

1. Pearson's Correlation Coefficient (r)

Most commonly used method. Measures linear correlation between two continuous variables.

r = Σ[(Xi - X̄)(Yi - Ȳ)] / √[Σ(Xi - X̄)² × Σ(Yi - Ȳ)²]
  • Assumes normal distribution of data
  • Sensitive to outliers
  • Best for continuous, normally distributed data

2. Spearman's Rank Correlation Coefficient (ρ)

Non-parametric method based on ranks rather than actual values. Used when data is ordinal or not normally distributed.

ρ = 1 - (6 × Σd²) / (n(n² - 1))
where d = difference between ranks of each pair
  • Robust to outliers
  • Measures monotonic relationships (not just linear)

3. Kendall's Tau (τ)

Another rank-based non-parametric method. Counts concordant and discordant pairs.

τ = (Concordant pairs - Discordant pairs) / [n(n-1)/2]
Types of Correlation — Scatter Plot Pattern

Impact of Outliers on Correlation

Pearson's r is highly sensitive to outliers — a single extreme value can significantly change the correlation value. Spearman's ρ is preferred when outliers are present.

What is Dimensionality Reduction?

Dimensionality reduction is the process of reducing the number of features (variables) in a dataset while retaining as much important information as possible. High-dimensional data is difficult to visualize, computationally expensive, and may suffer from the "curse of dimensionality".

Why is it Needed?

  • Reduces computation time and storage
  • Removes redundant or irrelevant features
  • Improves model accuracy by reducing noise
  • Makes data visualization possible (2D/3D)

Two Main Approaches

Feature Selection

Select a subset of the most relevant original features. Features not chosen are discarded.

Example: Keeping only "age" and "salary" from 10 features.

Feature Extraction

Transform original features into a new, lower-dimensional space. New features are combinations of old ones.

Example: PCA creates new components from combinations.

Principal Component Analysis (PCA) — Linear Algebra Approach

PCA uses linear algebra (matrices, eigenvectors, eigenvalues) to find the directions (principal components) of maximum variance in the data.

  1. Standardize the dataset (mean = 0, variance = 1)
  2. Compute the Covariance Matrix of features
  3. Find Eigenvectors and Eigenvalues of the covariance matrix
  4. Sort eigenvectors by their eigenvalues (descending)
  5. Select top k eigenvectors as principal components
  6. Project original data onto these components
PCA — Dimensionality Reduction Flow
📝 Key Formula: Covariance matrix C = (1/n) × Xᵀ × X. Eigenvectors of C give the principal components.

What is Factor Analysis?

Factor Analysis is a statistical technique used to reduce a large number of observed variables to a smaller number of unobserved latent factors that explain the patterns of correlation among the observed variables.

It assumes that observed variables are linear combinations of underlying hidden factors plus some error.

X = ΛF + ε
where X = observed variables,
Λ = factor loadings,
F = latent factors,
ε = error

Key Concepts

  • Factor Loadings: Correlation between observed variables and the factors. High loading means the variable is strongly related to that factor.
  • Communality: Proportion of variance in a variable explained by the common factors.
  • Factor Rotation: Rotating factors (e.g., Varimax) to make results more interpretable.
  • Eigenvalue: Indicates the amount of variance explained by each factor. Factors with eigenvalue > 1 are retained.

Steps of Factor Analysis

  1. Collect data and check suitability (KMO test, Bartlett's test)
  2. Compute correlation matrix
  3. Extract factors (using PCA or Maximum Likelihood)
  4. Determine number of factors (scree plot, eigenvalue > 1 rule)
  5. Rotate factors for better interpretation
  6. Interpret and name the factors

What is Regression?

Regression is a supervised learning technique used to model the relationship between a dependent variable (output) and one or more independent variables (inputs). It is used for prediction and forecasting.

1. Simple Linear Regression

Models the relationship between ONE independent variable (X) and a continuous dependent variable (Y) using a straight line.

Y = β₀ + β₁X + ε
Where β₀ = intercept,
β₁ = slope,
ε = error term

Example: Predicting house price based on area.

2. Multiple Linear Regression

Extension of simple linear regression with TWO or more independent variables.

Y = β₀ + β₁X₁ + β₂X₂ + ... + βₙXₙ + ε

Example: Predicting salary based on experience, education, and skills.

3. Ordinary Least Squares (OLS) Regression

OLS is the method used to estimate the coefficients (β values) in linear regression. It minimizes the sum of squared differences between observed and predicted values.

Minimize: Σ(Yᵢ - Ŷᵢ)² = Σ(Yᵢ - β₀ - β₁Xᵢ)²
  • OLS gives the Best Linear Unbiased Estimator (BLUE)
  • Assumptions: linearity, homoscedasticity, no multicollinearity, normality of errors

4. Logistic Regression

Used when the dependent variable is categorical (binary: 0/1, Yes/No). It predicts the probability of an outcome using the sigmoid function.

P(Y=1) = 1 / (1 + e^(-z)) where z = β₀ + β₁X₁ + β₂X₂ + ...

Example: Predicting whether an email is spam (1) or not (0).

5. Non-Linear Regression

Used when the relationship between variables is not linear (polynomial, exponential, logarithmic).

Y = β₀ + β₁X + β₂X² + ... + βₙXⁿ (Polynomial)
Regression Types Summary
📝 Exam tip: Distinguish OLS as a METHOD (for estimating coefficients), not a separate type of regression. Logistic regression output is a probability (0 to 1), not a continuous value.

What is Time Series Analysis?

A time series is a sequence of data points collected or recorded at successive points in time, usually at uniform intervals. Time Series Analysis involves methods for analyzing time series data to extract meaningful statistics, identify patterns, and forecast future values.

Examples: Daily stock prices, monthly rainfall, annual GDP growth, website traffic per hour.

Components of a Time Series

Trend (T)

Long-term increase or decrease in the data. E.g., population growth over decades.

Seasonality (S)

Regular, repeating patterns at fixed intervals. E.g., sales spike every December.

Cyclicity (C)

Irregular longer-term fluctuations related to economic cycles (not fixed period).

Irregularity / Noise (I)

Random variation that cannot be explained by the other components.

Components of Time Series

Time Series Analysis Techniques

1. Moving Average (MA)

Smooths out short-term fluctuations by averaging data over a sliding window of time periods. Used to identify trends.

MA(t) = (X(t) + X(t-1) + ... + X(t-n+1)) / n

2. Exponential Smoothing

Assigns exponentially decreasing weights to older observations. More recent data gets higher weight. Suitable for short-term forecasting.

S(t) = α × X(t) + (1-α) × S(t-1) where 0 < α < 1

3. ARIMA (AutoRegressive Integrated Moving Average)

Most powerful technique. Combines three components: AR (past values), I (differencing to make series stationary), MA (past forecast errors).

ARIMA(p, d, q)
Where p = AR order,
d = differencing,
q = MA order

4. Seasonal Decomposition

Decomposes time series into Trend, Seasonal, and Residual components to study each separately.

Additive model: Y(t) = T(t) + S(t) + R(t)
Multiplicative model: Y(t) = T(t) × S(t) × R(t)

5. Autocorrelation Analysis

Measures correlation of the series with its own past values (lags). Used to identify patterns and select ARIMA parameters.

📝 Exam tip: Always mention the 4 components (TSCI) and at least 3 techniques with brief explanation. ARIMA is the most important advanced technique to mention.

Q 10 Describe Normal, Binomial and Categorical distributions. / Differentiate MultinomialNB, BernoulliNB and GaussianNB. 3 M
Sum'24

1. Normal Distribution

A continuous probability distribution that is symmetric and bell-shaped around the mean (μ). Defined by mean (μ) and standard deviation (σ).

f(x) = (1/σ√2π) × e^[-(x-μ)²/2σ²]

Properties: Mean = Median = Mode; 68-95-99.7 rule (empirical rule). Used in GaussianNB classifier.

2. Binomial Distribution

A discrete probability distribution for the number of successes in n independent Bernoulli trials, each with probability p. Used in BernoulliNB (binary features).

P(X = k) = C(n,k) × p^k × (1-p)^(n-k)

3. Categorical / Multinomial Distribution

Extends binomial to multiple categories (more than 2 outcomes). Models count data or frequency of categorical features. Used in MultinomialNB (text classification, word counts).
